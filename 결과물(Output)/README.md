# 결과물(Output)

실제로 실행/배포하는 산출물만 모아둔 폴더입니다.

- `mac/OrgChartStudio.app` — macOS용 독립 실행파일(더블클릭 실행, Python 불필요)
- `windows/` — Windows용 `OrgChartStudio.exe` (로컬 빌드 또는 GitHub Actions CI 산출물,
  `windows/README.md` 참고)
- `샘플양식(Templates)/` — 바로 열어볼 수 있는 조직도 엑셀 템플릿·예시 데이터

소스 코드·테스트·빌드 스크립트 등 개발용 자료는 모두 `../실행관리(Ops-Run)/`에 있습니다.
