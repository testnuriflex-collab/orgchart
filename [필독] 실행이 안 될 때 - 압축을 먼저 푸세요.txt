★★★ 실행 전에 꼭 읽어 주세요 ★★★

"Failed to load Python DLL ... python312.dll" 오류가 뜬다면,
프로그램이 압축 파일(zip) 안에서 실행된 것입니다.

압축 프로그램(알집, 반디집, 윈도우 기본 등)으로 zip을 "열기"만 하면
진짜 폴더처럼 보이지만, 실제로는 임시 폴더에 일부 파일만 꺼내져서
프로그램이 실행되지 못합니다.

■ 올바른 실행 방법 (Windows)

  1. 내려받은 jogikdo.zip 파일 위에서 마우스 오른쪽 버튼 클릭
  2. "압축 풀기..." (또는 "모두 추출...") 선택
  3. 풀 위치를 "바탕 화면"으로 정하고 [압축 풀기] 버튼 클릭
  4. 압축이 다 풀리면 바탕 화면에 생긴 jogikdo 폴더 열기
  5. start_windows 폴더 안의 OrgChartStudio.exe 더블클릭

■ 제대로 되었는지 확인하는 방법

  폴더 창 위쪽 주소칸을 클릭했을 때
  - "Temp" 나 ".zip" 글자가 보이면 → 아직 압축이 안 풀린 상태입니다. 1번부터 다시.
  - "C:\Users\...\Desktop\jogikdo\..." 처럼 보이면 → 정상입니다.

  또한 OrgChartStudio.exe 바로 옆에 "_internal" 폴더가
  같이 보여야 합니다. (exe만 따로 복사하면 실행되지 않습니다.)

■ 그래도 안 되면

  1. jogikdo.zip 을 처음부터 다시 내려받은 뒤 위 순서를 반복해 주세요.
     (파일 이름 끝에 .zip 이외의 글자가 붙어 있으면 잘못 받아진 파일입니다.)
  2. 백신 프로그램의 "격리함"에 python312.dll 이 들어가 있지 않은지 확인해 주세요.
  3. 그래도 안 되면 구매처로 문의해 주세요. 오류 창 사진과 함께,
     실행한 폴더의 주소칸 사진을 보내 주시면 빠르게 도와드릴 수 있습니다.
